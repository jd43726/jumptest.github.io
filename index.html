CSS grid has two main concepts, defining the grid container itself and interacting with the grid items.

Grid container

The grid container is what defines a grid and is the parent element of the grid container. You cannot use CSS Grid without having at least one container. To make an element a container, simply add the display: grid to the element that will be the container.

.container {
   display: grid;
}
